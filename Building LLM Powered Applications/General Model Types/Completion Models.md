### About
* Raw definition: Takes a **single** input and produces an text output
	* Meaningless
* Extended meaning:
	* Model will try to continue the prompt according in a meaningful way
* Examples:
	* Generating summaries
	* Translations
	* Stories
	* Code
	* etc.
### Difference with Assistant/Chat Models
* Chat models expect multiple messages (e.g., a list)
* Completion models expect 1 input
